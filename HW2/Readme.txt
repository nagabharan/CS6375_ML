Machine Learning HW 2

The program contains necessary comments and solution for the regular AND BONUS question.

Steps to compile and run the java project

	1.Import the project in Eclipse

	2.To run the program select Run configuration with the following arguments in the args tab:

		- with the tree
			10 15 training_set#.csv validation_set#.csv test_set#.csv yes

		- without the tree
			10 15 training_set#.csv validation_set#.csv test_set#.csv no

The DT txt files contains the Decision tree generated using the specified format.
DTs are printed for before and after pruning for ID3-Entropy,ID3-Variance Impurity and Random Attribute.
